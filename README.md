# 📝 To-Do List Android

Um aplicativo de lista de tarefas simples e elegante desenvolvido em Java para Android. O projeto utiliza `ListView` com um `ArrayAdapter` customizado e um design moderno baseado em `CardView`.

## 🚀 Funcionalidades

*   **Adicionar Tarefas:** Digite uma tarefa e adicione-a instantaneamente à lista.
*   **Remover Tarefas:** Toque em qualquer item da lista para removê-lo.
*   **Interface Moderna:** Layout limpo com sombras, bordas arredondadas e feedback visual.

## 🛠️ Tecnologias Utilizadas

*   **Linguagem:** Java
*   **UI Layout:** XML (ConstraintLayout, LinearLayout)
*   **Componentes:** 
    *   `ListView` para exibição de dados.
    *   `CardView` para o design dos itens.
    *   `ArrayAdapter` customizado para gerenciamento da lógica da lista.

## 📱 Visual do Projeto

O app utiliza uma paleta de cores moderna com fundo cinza claro (`#F5F5F5`) e destaques em roxo (`#6200EE`), garantindo uma boa experiência de leitura e uso.

## 📂 Estrutura de Arquivos Principais

*   `MainActivity.java`: Controla a lógica principal e entrada de dados.
*   `MeuAdapter.java`: Gerencia a ponte entre os dados e o layout customizado.
*   `activity_main.xml`: Define a interface principal do usuário.
*   `item_lista.xml`: Define o design de cada cartão de tarefa na lista.
